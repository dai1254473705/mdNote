# 侧边栏与功能百宝箱使用指南

本文档详细介绍了侧边栏 (Sidebar) 的核心功能，特别是新增的“功能百宝箱”区域。

## 1. 功能百宝箱 (Function Toolbox)

为了提供更高效的快捷操作体验，我们将常用的系统级功能整合到了侧边栏的**功能百宝箱**区域。该区域位于搜索框下方，支持折叠与展开。

### 📅 日程清单 (Schedule)
*   **功能**: 快速管理您的待办事项和日程安排。
*   **交互**: 点击即可打开日程面板，支持添加、完成和删除事项。
*   **提醒**: 若有逾期未完成的事项，图标旁会出现红色状态提示。

### 🔑 密码管理器 (Password Manager)
*   **功能**: 安全存储和管理您的账号密码信息。
*   **安全性**: 所有数据加密存储于本地，确保隐私安全。
*   **交互**: 点击打开管理弹窗，支持分类查看、复制密码和新增条目。

### 🗑️ 回收站 (Trash)
*   **功能**: 存放您删除的笔记和文件夹，提供“后悔药”机制。
*   **交互**:
    *   **查看**: 点击打开回收站列表查看已删除项目。
    *   **还原**: 选中项目可将其还原至原位置。
    *   **彻底删除**: 清空回收站或删除单个项目将永久销毁数据。
*   **状态**: 若回收站内有文件，图标旁会显示红色计数提示。

---

## 2. 顶部工具栏 (Header Actions)

侧边栏顶部区域集成了最常用的全局操作：

*   **🏠 切换项目**: 左上角显示当前项目名称，点击可切换或重新打开其他项目文件夹。
*   **➕ 新建**: 支持下拉选择“新建笔记”或“新建文件夹”。
*   **🔄 刷新**: 强制重新加载当前文件树（适用于外部文件变更场景）。
*   **❓ 帮助**: 点击问号图标，快速查看系统帮助文档和快捷键说明。

## 3. 文件导航与管理

*   **🔍 搜索**: 全局搜索文件名和文件内容。
*   **⭐ 我的收藏**: 快速访问您标记为“收藏”的高频文件或文件夹。
*   **📂 文件树**: 展示完整的项目目录结构，支持新建、重命名、拖拽移动等操作。
