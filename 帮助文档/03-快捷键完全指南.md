# 知夏笔记 - 快捷键完全指南

掌握快捷键，让你的笔记效率提升 10 倍！

> 💡 **提示**：Windows/Linux 用户将 `⌘` (Command) 替换为 `Ctrl` 即可

---

## 📁 文件操作

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `⌘ + N` | 新建笔记 | 在当前目录创建新的 Markdown 文件 |
| `⌘ + S` | 保存笔记 | 保存当前笔记（也支持自动保存） |
| `⌘ + E` | 导出笔记 | 导出为 MD、HTML 或 PDF 格式 |
| `⌘ + O` | 打开文件 | 快速打开文件选择器 |

---

## 🏷️ 标签管理

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `⌘ + W` | 关闭当前标签 | 关闭当前打开的笔记标签 |
| `⌘ + Tab` | 下一个标签 | 切换到右侧的标签 |
| `⌘ + Shift + Tab` | 上一个标签 | 切换到左侧的标签 |
| `⌘ + 数字键 1-9` | 跳转标签 | 直接跳转到指定序号的标签 |

## 💡 高级技巧

### 快速插入

在行首输入以下字符，按空格键自动转换为格式：

- `#` + 空格 → 一级标题
- `##` + 空格 → 二级标题
- `-` + 空格 → 无序列表
- `1.` + 空格 → 有序列表
- `>` + 空格 → 引用块
- ` ` ` → 代码块

### 多光标编辑

- `⌘ + D` - 选择下一个相同的词
- `⌘ + Click` - 在点击位置添加光标
- `Option + Click + 拖动` - 列选择

### 快速跳转

- `⌘ + Click` - 跳转到定义
- `⌘ + [` - 返回上一个位置
- `⌘ + ]` - 前进到下一个位置

## 🎯 常见场景快捷操作

### 写笔记流程

1. `⌘ + N` - 新建笔记
2. 输入标题（`# 标题` + 空格）
3. `⌘ + S` - 保存（可选，自动保存）
4. `⌘ + B` / `⌘ + I` - 格式化文本
5. `⌘ + W` - 完成后关闭标签

### 查看和搜索

1. `⌘ + Shift + F` - 打开全局搜索
2. 输入关键词
3. 点击搜索结果打开笔记
4. `⌘ + G` - 查看下一个匹配项

### 导出分享

1. 打开要导出的笔记
2. `⌘ + E` - 打开导出菜单
3. 选择格式（MD/HTML/PDF）
4. 选择保存位置

## 🎨 自定义快捷键

知夏笔记支持自定义快捷键（即将推出）：

1. 打开设置（`⌘ + ,`）
2. 进入"键盘"选项卡
3. 点击要修改的功能
4. 按下新的快捷键组合
5. 保存设置

---

## 💾 快捷键配置文件

快捷键配置保存在应用配置文件中：

```json
{
  "shortcuts": {
    "newFile": "Cmd+N",
    "save": "Cmd+S",
    "toggleSidebar": "Cmd+B"
  }
}
```

1. **冲突检测** - 如果快捷键与系统快捷键冲突，应用快捷键优先

3. **记忆技巧** - 常用快捷键形成肌肉记忆后，效率大幅提升
