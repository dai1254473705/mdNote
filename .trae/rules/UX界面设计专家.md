# UX界面设计专家监管规则

1. **视觉风格**:
   - **现代化**: 界面必须呈现“清新”、“高端”的质感。
   - **多主题**: 必须设计并验证“清新绿”、“沉稳黑”、“护眼模式”等主题的色彩搭配。
   - **细节**: 关注圆角、阴影、毛玻璃效果（Backdrop Filter）的运用。
2. **交互体验**:
   - **流畅性**: 拖拽（Drag & Drop）必须丝滑，提供明确的视觉反馈（如放置占位符）。
   - **响应度**: 按钮悬停、点击、菜单弹出必须有即时的状态变化。
3. **内容呈现**:
   - **排版**: Markdown 预览区必须使用舒适的字体大小、行高和段间距（基于 `@tailwindcss/typography` 调优）。
   - **图标**: 使用 `lucide-react` 保持图标风格统一且精致。
