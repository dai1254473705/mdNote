# mdNote

一款基于 Electron + React + TypeScript 的现代化本地 Markdown 笔记应用。

> **注意**：本项目正在进行重构，最新开发文档请参考 [docs/最新需求文档.md](docs/最新需求文档.md)。

## 核心特性

- **本地优先**：所有数据存储在本地，拥有完全控制权。
- **Git 同步**：基于 `.github-notebook` 目录结构，支持与 GitHub 仓库无缝同步。
- **安全加密**：支持 AES-256-GCM 笔记加密。
- **现代化 UI**：基于 Tailwind CSS + Headless UI 打造，支持多主题预览（清新绿、沉稳黑等）。
- **极简体验**：三级目录限制，专注于写作本身。
